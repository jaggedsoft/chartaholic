#### Getting Started
```js
let chart = new Chartaholic( "#chart", { ticks: ohlc } );
let chart = new Chartaholic( "#chart", { theme: "blue", keys: { time: "time_open" }, ticks, title, usd:true, overlay, annotations } );
```
